# Contributing to BrightSteps

Thanks for contributing to BrightSteps.

## Development Setup

- Install Node.js LTS.
- Install `pnpm`.
- Install dependencies:

```bash
pnpm install
```

- Start development server:

```bash
pnpm dev
```

## Lint and Test

Use the following commands (or the closest equivalents in this repo):

```bash
pnpm lint
pnpm test
```

If a command is not implemented yet, mention that clearly in your PR notes.

## Pull Request Guidelines

- Keep PRs focused and small.
- Include a clear description of the problem and solution.
- Include screenshots/videos for UI changes.
- Prioritize accessibility (keyboard support, contrast, readable labels).
- Follow calm UI principles suitable for neurodivergent users: predictable layouts, minimal visual noise, clear feedback states.

## Content Contribution Rules

- Facts must be sourced and verifiable.
- Avoid political framing and advocacy content.
- Keep language literal, direct, and age-appropriate.
- Do not include medical advice, diagnosis, or treatment guidance.

## Security and Privacy

- Do not introduce analytics tracking or ad-tech.
- Do not add collection of child personal data.
- Keep BrightSteps local-first by default.
